# Markdown

Markdown es un lenguaje de marcado ligero (*lightweight markup language*) que te permite dar formato a texto plano usando caracteres simples. Lo que escribes se transforma automáticamente en texto con formato visual.

Es el estándar de facto para documentación técnica: lo usan GitHub, GitLab, Notion, Jupyter Notebooks, y decenas de herramientas más.
