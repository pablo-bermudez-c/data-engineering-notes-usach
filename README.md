[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

Este repositorio tiene la finalidad de resumir clases de un diplomado de ingeniería de datos y entregar otras herramientas informáticas que pueden ser de utilidad para el área.

## Contenido

El repositorio está organizado en dos secciones principales:

### Cursos del diplomado

Resúmenes y apuntes de los módulos del Diplomado en Ingeniería de Datos de la USACH.

| # | Módulo | Descripción |
|---|--------|-------------|
| 1 | [Fundamentos](diploma-courses/modulo-01-fundamentos/README.md) | IDEs, VS Code, asistentes de IA, SDD y TDD |

### Herramientas y prácticas

Tutoriales prácticos de herramientas complementarias, organizados en clases secuenciales.

| # | Tutorial | Clases | Descripción |
|---|----------|--------|-------------|
| 1 | [GitHub](tools-and-practices/github/README.md) | 8 | Control de versiones con Git y colaboración en GitHub |
| 2 | [Python - Ambientes Virtuales](tools-and-practices/python-ambientes-virtuales/README.md) | 5 | Ambientes virtuales y gestión de proyectos con uv |

## ¿Cómo usar este repositorio?

- Cada módulo o tutorial está organizado en **clases** (`clase-XX-tema/README.md`) que deben seguirse en orden.
- Cada clase incluye objetivos de aprendizaje, explicaciones con ejemplos y ejercicios.
- Los términos técnicos se presentan en español con su equivalente en inglés entre paréntesis, por ejemplo: "rama (*branch*)".
