# Documentación de Código

Documentar bien el código es tan importante como escribirlo. Un proyecto sin documentación es difícil de entender, mantener y compartir — incluso para tu yo del futuro.

Este módulo cubre dos herramientas fundamentales para documentar proyectos técnicos:

- **Markdown**: el formato de texto más usado en documentación técnica, READMEs y plataformas como GitHub.
- **Mermaid**: una herramienta para crear diagramas directamente desde texto, sin necesidad de software gráfico.

---

## Prerequisitos

- Conocimiento básico de cómo navegar carpetas y crear archivos.
- Tener un editor de texto instalado (se recomienda VS Code).
- Haber revisado el módulo de [GitHub](../github/) ayuda, pero no es obligatorio.
