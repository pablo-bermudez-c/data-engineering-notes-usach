## ¿Cómo usar estos tutoriales?

Cada módulo está organizado en **clases** que deben seguirse en orden. Cada clase incluye:

- Objetivos de aprendizaje
- Explicaciones con ejemplos prácticos
- Ejercicios para practicar

> Los términos técnicos se presentan en español con su equivalente en inglés entre paréntesis, por ejemplo: "rama (*branch*)".
