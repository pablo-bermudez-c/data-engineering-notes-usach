# Mermaid

Mermaid es una herramienta que permite crear diagramas a partir de texto. En lugar de usar software gráfico como Lucidchart o draw.io, escribes una descripción del diagrama y Mermaid lo genera automáticamente.

Está integrado nativamente en GitHub, GitLab, Notion y VS Code (con extensión), lo que lo convierte en una herramienta ideal para incluir diagramas directamente en la documentación de un proyecto.
